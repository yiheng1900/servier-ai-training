# Servier AI Training Site

Static site for the "AI赋能医生培训体系" page.

## Local preview

Run a simple static server from this directory, for example:

```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000`.

## Deployment

This repository can be deployed in two ways:

- GitHub Pages from the `main` branch root
- Render via the included `render.yaml`
